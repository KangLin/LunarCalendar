Use QT write lunar calendar.


- Windows
- Linux、Unix
- Android
- Mac os
- IOS

------------------------------------------------

[![Build status](https://ci.appveyor.com/api/projects/status/p5vhmmbuql9fyfpl/branch/master?svg=true)](https://ci.appveyor.com/project/KangLin/lunarcalendar/branch/master)

------------------------------------------------

### Compile
- Create and enter the build directory

        git clone --recursive https://github.com/KangLin/LunarCalendar.git
        cd LunarCalendar
        mkdir build

- Compile

        qmake LunarCalendar.pro
        make install

------------------------------------------------
### Use
- Direct source code. Is a QT project, directly introduces LunarCalendar.pri

         Include(LunarCalendar.pri)
